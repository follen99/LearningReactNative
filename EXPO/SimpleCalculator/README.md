# Simple calculator app
Semplice app "calcolatrice"
## Struttura

```
SimpleCalculator/
│
├── assets/                   # Immagini, icone, font (se ne usi)
│
├── src/                      # Tutto il codice della tua app
│   ├── components/           # Componenti riutilizzabili (Display, Button, Keypad)
│   ├── screens/              # Le schermate: CalculatorScreen, AboutScreen, ecc.
│   ├── navigation/           # Stack navigator o tab navigator
│   ├── utils/                # Funzioni di supporto (es. calcolo)
│   └── styles/               # (facoltativo) Stili comuni
│
├── App.js                    # Entry point della tua app
├── app.json                  # Configurazione Expo
├── package.json
└── babel.config.js
```
